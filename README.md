# Image Clustering Practice

## Setup

1: Create venv environment

```sh
python -m venv venv
```

2: Activate venv

```sh
source venv/bin/activate
```

3: Install requirements

```sh
pip install -r requirements.txt
```

4: Put images

```sh
mkdir imageset
cp /path/to/images/* imageset/
```

## Run

```sh
python main.py
```
